# 🛒 Mini Shopping Cart (Python)

A simple command-line shopping cart built using Python.

This project was created as part of my Python learning journey to practice core programming concepts by building a real mini project instead of isolated exercises.

## ✨ Features

- Add items to the shopping cart
- View all items in the cart
- Remove items from the cart
- Detect an empty cart
- Handle invalid menu choices
- Menu-driven interface
- Runs until the user chooses to exit

## 🧠 Concepts Used

- Functions
- Lists
- User Input
- While Loops
- Conditional Statements (`if`, `elif`, `else`)
- List Methods
  - `append()`
  - `remove()`
- Basic Program Structure

## ▶️ How to Run

1. Clone the repository.
2. Run the Python file.

- Modifying lists using `append()` and `remove()`
- Organizing code using multiple functions
- Debugging logical errors
